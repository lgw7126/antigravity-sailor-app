# 🧭 세일러 (Sailor) - 쇼핑 할인 정보 검색 앱 MVP 프로토타입

> **현명한 쇼핑을 위한 최저가 정보 나침반, 세일러(Sailor)**  
> G마켓 온라인 최저가부터 내 주변 동네 마트의 실시간 오프라인 세일 정보까지, 직관적인 거리/할인율 필터 조작으로 최적의 핫딜을 탐색하는 모바일 우선(Mobile-first) 웹 앱 프로토타입입니다.

---

## 🔍 프로젝트 개요
실제 API 연동 전, 프론트엔드의 화면 레이아웃과 필터링/정렬 로직의 동작성을 완벽히 검증하기 위해 설계된 **고해상도 MVP 프로토타입**입니다. 데스크톱과 모바일 환경을 모두 고려한 유연한 레이아웃과 실제 앱을 사용하는 듯한 마이크로 인터랙션을 제공합니다.

---

## ✨ 핵심 기능 및 디자인 특징

### 1. 데스크톱용 모바일 디바이스 시뮬레이터 (Device Frame)
- 데스크톱 해상도에서 확인 시 화면 중앙에 세련된 스마트폰 기기 프레임이 나타납니다.
- 기기 상단의 스피커 노치, 카메라 홀, 실시간으로 갱신되는 상태 표시줄(배터리, 와이파이, 현재 시각) 등을 묘사하여 실제 앱을 조작하는 느낌을 극대화했습니다.
- 모바일 해상도에서는 자동으로 기기 프레임이 숨겨지며 화면 전체를 꽉 채우는 반응형 레이아웃으로 전환됩니다.

### 2. 통합 검색창 (Header)
- 상단 헤더에 상품명이나 마트명(예: "신라면", "이마트")을 입력할 수 있는 돋보기 아이콘 디자인의 검색 필드입니다.
- 검색어가 입력되면 자동으로 지우기 버튼(X)이 나타나며, 타이핑과 동시에 리스트가 즉시 필터링됩니다.

### 3. 온/오프라인 전환 탭 (iOS 스타일 토글)
- **지마켓 온라인 최저가**: 온라인 쇼핑몰(지마켓, 지마켓 스마일배송 등)의 전국 배송 딜을 보여줍니다. (거리 필터 비활성화)
- **내 주변 오프라인 세일**: 내 위치 반경에 위치한 대형마트 및 편의점(이마트, 홈플러스, GS25, CU 등)의 현장 세일 정보를 보여줍니다. (거리 필터 활성화)
- 토글 전환 시 슬라이더 배경이 부드러운 애니메이션 효과와 함께 이동합니다.

### 4. 직관적인 다중 필터바 (Filter Bar)
- **거리별 필터 (오프라인 전용)**: [전체 / 1km 이내 / 3km 이내 / 5km 이내]의 알약 모양 버튼 그룹입니다. 클릭 시 내 위치로부터의 거리를 계산하여 조건 이하의 상품만 실시간으로 렌더링합니다.
- **정렬 필터**: [기본순 / 할인율 높은 순 / 최저가 순] 드롭다운을 제공하여 실시간 정렬 연산을 제공합니다.

### 5. 모바일 최적화 카드 UI (Card List)
- 상품의 출처(머천트)별 시그니처 색상을 반영한 커스텀 배지를 표시합니다.
- 원래 가격은 취소선 처리되고 세일 가격이 굵게 강조됩니다.
- 할인율은 붉은색의 대형 폰트 배지로 우측에 시각적으로 강조 처리됩니다.
- 오프라인 모드에서는 우측 상단에 나와의 거리(예: 0.3km)가 나침반 화살표 아이콘과 함께 표시됩니다.

### 6. 상세 보기 모달 및 사용자 인터랙션 (Detail Modal & Toast)
- 할인 카드 클릭 시, 부드러운 스케일 및 페이드 애니메이션과 함께 상세 정보 팝업 모달이 뜹니다.
- 모달 내에서 **즐겨찾기(하트) 등록 및 해제** 기능이 작동하며 상태가 로컬 스토리지에 유지됩니다.
- **구매하러 가기** 버튼을 누르면 외부 링크 이동을 모사하는 세련된 하단 팝업 토스트 알림이 실행됩니다.

- **Markup**: `HTML5` (Semantic Elements 활용)

## 🚀 사용법 및 동작 확인

1. **프로젝트 실행**:
   - `index.html` 파일을 더블 클릭하여 아무 최신 웹 브라우저(Chrome, Safari, Edge 등)에서 바로 실행합니다.
   
2. **검색 기능 테스트**:
   - 상단 검색창에 **"신라면"** 또는 **"화장지"**를 입력하고 실시간으로 카드가 필터링되는지 확인합니다.
   - 우측의 **(X) 버튼**을 눌러 검색어를 한 번에 비워봅니다.

3. **온/오프라인 모드 전환**:
   - 상단 토글의 `내 주변 오프라인 세일`을 선택해 봅니다. 아래 카드 목록이 오프라인 마트 정보로 즉시 전환되고 중단 거리 필터 영역이 선명하게 활성화됩니다.
   - 반대로 `지마켓 온라인 최저가`를 누르면 거리 필터가 부드럽게 불투명(Blur) 처리되며 비활성화됩니다.

4. **오프라인 거리 필터 테스트**:
   - 오프라인 모드에서 거리 필터의 **"1km 이내"** 버튼을 눌러봅니다. 나와의 거리가 `0.3km`, `0.8km` 등 1km 미만인 상품만 선별됩니다.
   - **"3km 이내"** 또는 **"전체"**를 눌러 반경 기준을 변경해 봅니다.

5. **정렬 필터 조합**:
   - 정렬 선택 상자에서 **"최저가 순"** 또는 **"할인율 높은 순"**을 고르면 실시간으로 화면의 카드들이 재정렬됩니다.

6. **상세 보기 및 토스트 테스트**:
   - 마음에 드는 상품 카드를 클릭하여 팝업 모달을 엽니다.
   - 모달 하단 좌측의 **하트(♡) 버튼**을 눌러 즐겨찾기에 등록하고 해제할 때 화면 하단에 나타나는 토스트 메시지를 확인합니다.
   - **구매하러 가기** 버튼을 눌러 해당 쇼핑몰의 사이트(지마켓/이마트/홈플러스는 해당 상품 검색 결과 페이지, GS25/CU는 공식 웹사이트)가 새 창/새 탭에서 실제로 열리는지 확인합니다.
